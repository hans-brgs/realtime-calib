**realtime-calib** now has a public home. This site collects everything you need
to install the app, run a calibration, and understand the method — plus the
project's open-source licensing.

<!-- truncate -->

This first release of the documentation covers:

- **Getting started** — install with Docker or `uv`, and a quickstart.
- **Guides** — the operator workflow, step by step.
- **Reference** — the Caliscope-compatible configuration format and the HTTP API.
- **Research** — the method, grounded on Caliscope and OpenCV, and how to cite.

The project is licensed under **AGPL-3.0**, with a commercial license available
for proprietary use. See the [license page](/docs/open-source/license) for
details.

More to come — this space will host release notes and project updates.
