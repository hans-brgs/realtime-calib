The AGPL's network copyleft is not compatible with keeping your own code
proprietary. If you want to:

- embed realtime-calib in a **closed-source / proprietary** product, or
- offer it as a **hosted service without** releasing your source, or

classic dual-licensing model (as used by MongoDB, Grafana, Nextcloud and others):
the same code is AGPL for everyone, and a separate commercial license removes the
